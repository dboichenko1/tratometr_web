# Тратометр — веб-версия

Офлайн-трекер личных финансов. Чистая статика (HTML + CSS + один JS-файл),
без сборки, без зависимостей, без сети и без аналитики. Устанавливается на
домашний экран iPhone/Android как обычное приложение и работает полностью
без интернета.

**Возможности:** операции (расход / доход / перевод), счета и категории,
балансы, разбивка по категориям (пончик), графики динамики, спидометры
(лимит трат, целевой доход, баланс месяца), регулярные операции, тёмная тема
с акцентами, резервная копия в JSON/CSV.

## Приватность

Все данные хранятся только на устройстве (IndexedDB + резервная копия
localStorage). Нет аккаунта, сервера, облака и каких-либо сетевых запросов —
приложение ничего и никуда не отправляет. Резервная копия (JSON/CSV)
создаётся локально как файл и остаётся у вас.

## Запуск на GitHub Pages

1. Создайте публичный репозиторий и положите содержимое этой папки в его корень.
2. **Settings → Pages → Build and deployment → Source: Deploy from a branch**,
   ветка `main`, папка `/ (root)`. Сохраните.
3. Через минуту приложение откроется по адресу `https://<ваш-логин>.github.io/<репозиторий>/`.

Все пути в проекте относительные, поэтому он работает и из корня домена, и из
подпапки project-страницы — настраивать ничего не нужно. Файл `.nojekyll`
отключает обработку Jekyll.

## Установка на iPhone (домашний экран)

1. Откройте адрес Pages в **Safari**.
2. Кнопка «Поделиться» → **«На экран „Домой"»** → «Добавить».
3. Запускайте с иконки — приложение открывается в полноэкранном режиме и
   работает офлайн. Срок жизни не ограничен (это не подпись iOS — ничего
   переподписывать не нужно).

> Совет: перед сменой телефона или удалением иконки сделайте резервную копию
> (раздел **Ещё → Резервная копия → Сохранить копию**) и восстановите её на
> новом устройстве.

## Локальный просмотр

Service worker требует `http(s)`, поэтому открывать через `file://` нельзя —
поднимите любой локальный сервер в этой папке:

```bash
python3 -m http.server 8080
# затем откройте http://localhost:8080
```

## Структура

```
index.html              каркас, мета-теги PWA, регистрация service worker
app.css                 стили (тёмная тема по умолчанию)
app.js                  всё приложение: данные, логика, экраны
manifest.webmanifest    манифест PWA (иконки, имя, цвета)
sw.js                   service worker — кеширование для офлайна
icons/                  иконки приложения
.nojekyll               отключает Jekyll на GitHub Pages
```

## Обновление версии

При изменении файлов поднимите версию кеша в `sw.js` (константа `VERSION`),
иначе у уже установивших приложение останется старая закешированная копия.
